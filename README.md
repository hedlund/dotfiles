# dotfiles

Simply my dotfiles, primarily aimed at being used on a Fedora Silverblue system.

## Initiate Silverblue

```sh
./init/silverblue.sh
```

## Bootstrap dotfiles

From within a Distrobox container, run:

```sh
./bootstrap.sh
```
